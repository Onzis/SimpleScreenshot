# SnapFlow

Минималистичный скриншотер для Windows : захват области экрана,
копирование в буфер, сохранение в PNG и глобальные горячие клавиши. Работает
в системном трее.

## Возможности

- Захват произвольной области экрана с затемнением фона (аккуратный «вырез» через `Path` + `EvenOdd`).
- Интерактивное выделение: перетаскивание области и изменение размера маркерами по углам и сторонам.
- Компактная панель инструментов под выделением: **Копировать**, **Сохранить**, **Закрыть**.
- Асинхронное копирование и сохранение — интерфейс не подвисает.
- Глобальная горячая клавиша вызова (по умолчанию **Shift + Alt + S**), настраивается.
- Значок и меню в системном трее в тёмном оформлении.
- Настройки сохраняются между запусками.
- Проверка новой версии и автообновление с GitHub Releases.

## Горячие клавиши

| Действие                     | Клавиши                     |
|------------------------------|-----------------------------|
| Сделать скриншот области     | `Shift + Alt + S` (глобально, настраивается) |
| Копировать выделение         | `Ctrl + C` (в режиме выделения) |
| Сохранить выделение          | `Ctrl + S` (в режиме выделения) |
| Отмена / закрыть             | `Esc`                       |

## Использование

1. Запустите приложение — иконка появится в системном трее.
2. Нажмите `Shift + Alt + S` (или двойной клик по иконке в трее).
3. Выделите область мышью. При необходимости перетащите её или измените размер маркерами.
4. Нажмите **Копировать** / **Сохранить** на панели или используйте `Ctrl + C` / `Ctrl + S`.

### Меню в трее

- **Скриншот области** — запустить захват.
- **Настройки…** — изменить горячую клавишу вызова.
- **Проверить обновления** — проверить наличие новой версии.
- **Выход** — закрыть приложение.

### Обновления

При запуске SnapFlow автоматически проверяет наличие новой версии на
[GitHub](https://github.com/Onzis/SnapFlow/releases). Если версия доступна,
приложение предложит скачать и установить её — новый `.exe` заменит текущий,
после чего программа перезапустится.

Проверить вручную: трей → **Проверить обновления**.

> Для работы автообновления в каждый релиз на GitHub должен быть приложен
> файл `SnapFlow.exe` (asset), а тег релиза — содержать номер версии, например `v1.3.0`.

### Смена горячей клавиши

Трей → **Настройки…** → кликните по полю и нажмите нужную комбинацию → **Сохранить**.
Комбинация применяется сразу. Настройки хранятся в:

```
%AppData%\SnapFlow\settings.json
```

## Сборка и запуск

Требуется **.NET 8 SDK** (Windows).

```powershell
# сборка
dotnet build -c Release

# запуск
dotnet run -c Release
```

Готовый исполняемый файл: `bin\Release\net8.0-windows\SnapFlow.exe`.

## Структура проекта

| Файл                     | Назначение                                            |
|--------------------------|-------------------------------------------------------|
| `App.xaml` / `.cs`       | Точка входа, трей, регистрация горячей клавиши        |
| `OverlayWindow.xaml`/`.cs` | Оверлей: затемнение, выделение, drag/resize, панель |
| `ScreenshotHelper.cs`    | Захват экрана, обрезка, асинхронный буфер/сохранение  |
| `HotkeyManager.cs`       | Регистрация глобальной горячей клавиши (Win32)        |
| `AppSettings.cs`         | Загрузка/сохранение настроек                          |
| `SettingsWindow.xaml`/`.cs` | Окно смены горячей клавиши                         |
| `DarkMenuRenderer.cs`    | Тёмное оформление меню трея                           |
| `app.ico`                | Иконка приложения                                     |

## Технологии

- .NET 8 · WPF
